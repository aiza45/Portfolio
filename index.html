KMEAN

This code applies three clustering techniques—K-Means, Hierarchical Clustering, and DBSCAN—on scaled data to group similar points. K-Means performance is evaluated using silhouette score and inertia, while DBSCAN is checked for meaningful clusters excluding noise points. The results are visualized using scatter plots for K-Means and DBSCAN, and a dendrogram for hierarchical clustering. This allows comparison of clustering quality and structure across different algorithms.
